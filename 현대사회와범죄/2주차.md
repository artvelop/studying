# 포스트모더니즘과 범죄현상

+ 모던적 기획

    + 선천적으로 평등하고, 자율적 존재, 누구든지 행복 추구, 이성적 판단과 과학적 노력으로 합리적 범과 제도 구성, 현실 세계의 모두가 조화롭게 살 수 있는 세상

    + 모순

        + 출생의 우연성

+ 포스트모더니즘

    + 모던적 기획의 모순으로 새롭게 형성된 현대 사회의 문화적 특징
    + 이질성/다양성의 관용

+ 도덕적 감성주의와 개인화된 사회정의

    + 과거판단기준: 개인 외부의 불변한 사회질서나 사물의 본질
